Question 10: Determinant Properties
(a) Three Properties of Determinants
Property 1: Multiplicative Property
det(AB) = det(A) · det(B)
Property 2: Determinant of Transpose
det(Aᵀ) = det(A)
Property 3: Scalar Multiplication
For an n×n matrix: det(kA) = kⁿ·det(A)
(b) Effect of Row Swap
When two rows of a matrix are swapped, the determinant changes sign.
If B is obtained from A by swapping two rows, then:
det(B) = -det(A)
(c) Why Nonzero Determinant for Inverse
The inverse formula is: A⁻¹ = 1det(A) · adj(A)
If det(A) = 0, then 1/det(A) is undefined (division by zero).
Additionally, det(A) = 0 means the matrix is singular (rows/columns are linearly dependent),
indicating the transformation is not invertible.
(d) Calculate det(AB) and det(A⁻¹)
Given: det(A) = 5 and det(B) = -2
Finding det(AB):
det(AB) = det(A) · det(B) = 5 · (-2) = -10
Finding det(A⁻¹):
Since A·A⁻¹ = I and det(I) = 1:
det(A) · det(A⁻¹) = 1
det(A⁻¹) = 1/det(A) = 1/5
det(AB) = -10
det(A⁻¹) = 1/5 = 0.2
